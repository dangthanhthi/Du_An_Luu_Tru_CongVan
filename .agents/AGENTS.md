# Project Rules

- **Tự động hoàn thành & Chạy dự án:** Luôn thực hiện toàn bộ các bước triển khai dự án hoàn chỉnh. Không yêu cầu người dùng chạy các lệnh khởi động (như chạy database, backend API, frontend dev server). Tự động chạy tất cả các dịch vụ cần thiết ở chế độ nền (background task) để người dùng có thể truy cập ngay lập tức qua trình duyệt.
